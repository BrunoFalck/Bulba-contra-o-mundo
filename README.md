# Bulba-contra-o-mundo
Jogue como Bulbassauro para derrotar todos os Patrats no seu caminho

Jogo desenvolvido a partir da API canvas do HTML5 e JavaScript.
O jogo foi desenvolvido com base no livro "Desenvolva jogos com HTML5 Canvas e JavaScript", de Éderson Cássio para a disciplina de Aplicações Frontend da faculdade Projeção.

Desenvolvedores: https://github.com/BrunoFalck e https://github.com/NFalcao.

Todo o áudio do jogo é feito com a voz dos desenvolvedores.
Divirta-se!
